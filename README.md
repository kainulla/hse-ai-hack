# HSE AI Analytics

AI-аналитика для системы охраны труда в нефтегазовой отрасли Казахстана.

## Возможности

- **Дашборд аналитики** — статистика происшествий, фильтрация по организации, типу, периоду
- **Предиктивная модель** — прогноз инцидентов на 3/6/12 месяцев (Prophet) с доверительными интервалами
- **Система алертов** — 4 уровня оповещений по данным Карт Коргау (красный, оранжевый, жёлтый, зелёный)
- **AI-рекомендации** — генерация рекомендаций через LLM (OpenAI) с rule-based fallback
- **Экономический эффект** — расчёт ROI от внедрения AI-модуля в людях и деньгах
- **REST API** — FastAPI с автоматической Swagger-документацией

## Быстрый старт

```bash
# 1. Создать виртуальное окружение
python3 -m venv .venv
source .venv/bin/activate

# 2. Установить зависимости
pip install -r requirements.txt

# 3. (Опционально) Настроить OpenAI для AI-рекомендаций
cp .env.example .env
# Отредактировать .env и указать OPENAI_API_KEY

# 4. Сгенерировать синтетические данные
python data/generate_data.py

# 5. Запустить дашборд
streamlit run app.py

# 6. (Опционально) Запустить API-сервер
uvicorn src.api:app --reload
```

## API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/orgs` | Список организаций |
| GET | `/api/incidents` | Происшествия (фильтры: org_id, type, date_from, date_to) |
| GET | `/api/korgau` | Карты Коргау (фильтры: org_id, category, date_from, date_to) |
| GET | `/api/forecast/{horizon}` | Прогноз на N месяцев |
| GET | `/api/risk-zones` | Топ зон риска |
| GET | `/api/alerts` | Активные алерты |
| GET | `/api/recommendations/{org_id}` | Рекомендации для организации |
| GET | `/api/economics` | Экономический эффект |
| GET | `/api/correlation` | Корреляция нарушений и инцидентов |

Swagger UI доступен по адресу: `http://localhost:8000/docs`

## Демо-сценарий

1. Откройте **Обзор** — просмотрите KPI и общую статистику
2. Выберите в фильтре организацию с высоким риском (например, ТОО «СтройМонтажСервис»)
3. Перейдите в **Прогнозы** — посмотрите прогноз инцидентов и метрики бэктеста
4. Откройте **Алерты Коргау** — обратите внимание на красный алерт
5. Перейдите в **Рекомендации** — просмотрите сгенерированные рекомендации
6. Завершите на **Экономический эффект** — покажите предотвращённые НС и экономию ~130 млн ₸

## Данные

Синтетические данные генерируются скриптом `data/generate_data.py`:
- **incidents.csv** — 686 происшествий за 3 года (2023-2025)
- **korgau_cards.csv** — 1686 карт Коргау за 3 года
- **organizations.json** — 8 организаций с профилями риска

Данные содержат сезонность (зимние и летние пики) и корреляцию между нарушениями Коргау и последующими инцидентами.

## Стек технологий

- **Python 3.11+**
- **Streamlit** — интерактивный дашборд
- **FastAPI** — REST API
- **Prophet** — прогнозирование временных рядов
- **Plotly** — визуализация
- **SQLite** — хранение данных
- **OpenAI API** — генерация рекомендаций (опционально)
